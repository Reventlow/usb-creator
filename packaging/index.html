usb-creator — package repository

Self-hosted repository for usb-creator (currently v__VERSION__), a single-file bash tool that writes verified Linux installer ISOs to USB sticks.

Signing key: __FPR__ (repo-key.asc)

Debian / Ubuntu (apt)

curl -fsSL __REPO_URL__/repo-key.asc | sudo tee /usr/share/keyrings/usb-creator.asc >/dev/null
echo "deb [signed-by=/usr/share/keyrings/usb-creator.asc] __REPO_URL__/apt stable main" | sudo tee /etc/apt/sources.list.d/usb-creator.list
sudo apt update && sudo apt install usb-creator

Fedora / openSUSE (dnf / zypper)

sudo curl -fsSL -o /etc/yum.repos.d/usb-creator.repo __REPO_URL__/rpm/usb-creator.repo
sudo dnf install usb-creator

Arch / CachyOS / EndeavourOS (pacman)

curl -fsSL __REPO_URL__/repo-key.asc | sudo pacman-key --add -
sudo pacman-key --lsign-key __FPR__
printf '[usb-creator]\nServer = __REPO_URL__/arch\n' | sudo tee -a /etc/pacman.conf
sudo pacman -Sy usb-creator

Packages are built and this image is assembled by GitHub Actions on every release; the repository metadata is GPG-signed with the key above.
